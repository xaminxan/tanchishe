# 微信小程序贪吃蛇游戏

这是一个可以在微信小程序上运行的经典贪吃蛇游戏。

## 项目结构

```
snake-game/
├── app.js                 # 小程序入口文件
├── app.json              # 小程序配置文件
├── project.config.json   # 项目配置文件
├── sitemap.json          # 站点地图配置
└── pages/
    └── index/
        ├── index.js      # 游戏逻辑
        ├── index.wxml    # 页面结构
        └── index.wxss    # 样式文件
```

## 功能特点

- 🎮 经典贪吃蛇玩法
- 📱 适配移动端触摸操作
- 🎯 虚拟方向键控制
- 🏆 最高分记录（本地存储）
- ⏸️ 暂停/继续功能
- 🎨 精美的游戏画面（蛇眼睛、苹果等）
- 💡 碰撞检测（墙壁和自身）

## 部署步骤

### 1. 下载微信开发者工具
访问 [微信开发者工具官网](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装

### 2. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择本项目目录 `/workspace/snake-game`
4. 填写你的 AppID（如果没有，可以选择测试号）
5. 点击"导入"

### 3. 编译运行
- 点击工具栏的"编译"按钮
- 在模拟器中即可看到游戏效果

### 4. 真机预览
1. 点击工具栏的"预览"按钮
2. 使用微信扫描二维码
3. 在手机上体验游戏

## 游戏操作

- **开始游戏**：点击中间的"开始游戏"按钮
- **控制方向**：使用屏幕上的方向键（↑↓←→）
- **暂停游戏**：游戏过程中点击"暂停"按钮
- **重新开始**：游戏结束后点击"重新开始"按钮

## 游戏规则

1. 蛇会自动向前移动
2. 吃到红色苹果会增加分数（每个10分）
3. 蛇身会变长
4. 撞到墙壁或自己的身体则游戏结束
5. 不能直接反向移动（如正在向右时不能直接向左）

## 自定义配置

你可以在 `pages/index/index.js` 中修改以下参数：

```javascript
data: {
  canvasWidth: 300,      // 画布宽度
  canvasHeight: 300,     // 画布高度
  gridSize: 15,          // 网格大小（蛇的大小）
  speed: 200,            // 游戏速度（毫秒/帧，越小越快）
}
```
