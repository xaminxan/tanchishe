# 微信小程序贪吃蛇游戏

## 项目说明
这是一个完整的微信小程序贪吃蛇游戏，使用原生 Canvas API 开发，无需任何游戏引擎。

## 文件结构
```
tanchishe/
├── app.js                 # 小程序入口
├── app.json              # 小程序配置
├── sitemap.json          # 站点地图配置
├── project.config.json   # 项目配置文件
└── pages/
    └── index/
        ├── index.wxml    # 页面结构
        ├── index.wxss    # 页面样式
        ├── index.js      # 游戏逻辑
        └── index.json    # 页面配置
```

### 1. 复制项目到本地
将 `/workspace/tanchishe` 文件夹中的所有文件复制到你的本地目录：

### 2. 下载微信开发者工具
- 访问官网：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 下载并安装稳定版

1. 打开微信开发者工具
2. 点击 "+" 号或 "导入项目"
3. 选择项目目录：`C:\Users\xaminxan\Desktop\tanchishe`
4. AppID 选择 "测试号"（无需注册正式账号）

### 4. 编译运行
- 工具会自动编译项目

- 如遇白屏，点击工具栏的 "编译" 按钮

## 游戏操作

### 电脑端（开发者工具）
- 键盘方向键控制蛇的移动
- 或点击屏幕上的虚拟方向键

### 手机端
- 点击 "预览" 按钮生成二维码
- 微信扫码在手机上玩
- 使用屏幕上的虚拟方向键控制

## 游戏规则
- 吃到红色苹果得 10 分
- 蛇身会随着得分增加而变长
- 速度会逐渐加快
- 撞墙或撞到自己则游戏结束
- 最高分会自动保存

## 常见问题

### Q: 为什么不用 Cocos Creator？
A: 贪吃蛇逻辑简单，用原生 Canvas API 更轻便：
- 包体积小（仅几十 KB）
- 启动快，无需加载庞大引擎
- 代码量少，开发简单
- 零依赖，只需微信开发者工具

### Q: 真机预览看不到画面？
A: 检查以下几点：
1. 确保手机和电脑在同一 WiFi
2. 在开发者工具中勾选 "不校验合法域名"
3. 重新编译项目

### Q: 如何修改游戏难度？
A: 在 `pages/index/index.js` 中修改：
- `speed`: 初始速度（毫秒），越小越快
- `gridSize`: 网格大小，影响蛇和食物的尺寸
